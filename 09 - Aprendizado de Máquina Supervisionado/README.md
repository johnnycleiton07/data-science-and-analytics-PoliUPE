# Aprendizado de Máquina Supervisionado

Conceito introdutório de Aprendizado Supervisionado com atividades práticas focadas na construção de modelos preditivos a partir de dados rotulados. A disciplina abordou algoritmos clássicos como o K-Nearest Neighbors (KNN), além de técnicas estatísticas como a Regressão Logística, explorando tarefas de classificação e regressão. Também foram discutidos conceitos fundamentais como treinamento e teste de modelos, avaliação de desempenho (acurácia, precisão, recall, F1-score), overfitting, validação cruzada e boas práticas para seleção e ajuste de modelos.
